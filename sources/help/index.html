ScriptGui - Help

O objetivo principal desta aplicação é o trabalho com os documentos simples, como txt, json, md, entre outros.

Ferramentas && Atalhos

Abaixo são todas as ferramentas e sua funcionalidade:

  1. Abrir - Abrirá um arquivo de texto presente na sua máquina
  2. Novo - Criara um arquivo
  3. Salvar - Salvará o arquivo em foco
  4. Salvar Como - Salvará uma copia arquivo em foco
  5. Fechar - Fechará o arquivo
  6. Rodar Script - Rodará o script solicitado(se for script)
Atalhos:
  • Abrir - Ctrl + O
  • Novo - Ctrl + N
  • Salvar - Ctrl + S
  • Salvar Como - Ctrl + shift + S
  • Fechar - Ctrl + p
  • Rodar Script - Ctrl + 5

Segredinho

Se apertar ALT + S, abrirá um arquivo de configuração, nele poderá configurar o app e até mesmo mudar a aparência dele.
