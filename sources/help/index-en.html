ScriptGui - Help

The main purpose of this application is to work with simple documents such as txt, json, md, among others.

Tools && Shortcuts

Below are all the tools and their functionality:

  1. Open - Will open a text file present on your machine
  2. New - Will create a file
  3. Save - Will save the hover file
  4. Save As - Will save a copy of hover file
  5. Close - Will close the file
Shortcuts:
  • Open - Ctrl + O
  • New - Ctrl + N
  • Save - Ctrl + S
  • Save As - Ctrl + shift + S
  • Close - Ctrl + p

Secret

If you press ALT + S, it will open a configuration file where you can configure the app and even change its appearance.
